# MERN User Management System

## 🚀 Live Demo
https://mern-crud-auth-three.vercel.app

## 📌 Features
- User registration and login  
- JWT-based authentication with token stored in localStorage  
- Display all users on dashboard after login  
- Create new users  
- Update existing users  
- Delete users  
- Secure backend integration with authentication and image upload  
- Responsive UI  

## 🛠️ Tech Stack
- MERN Stack (MongoDB, Express.js, React.js, Node.js)  
- Authentication: JWT (stored in localStorage)  
- API Calls: Fetch API (or Axios if used)
